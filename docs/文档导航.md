# 文档导航索引

> **最后更新**: 2024-04-18

---

## 使用说明

在开发过程中：
- **需要时可以查找对应文档**
- **不要擅自猜测**需求或实现方式，不确定时先看文档
- **严禁过度开发**，只做文档里明确要求的功能

需要参考文档时，请明确说明，例如：
- "看一下需求文档里的验收标准"
- "参考架构文档中的数据模型部分"

## 各文档详情

### 1. 需求文档 (`01-需求文档.md`)

**核心用途**: 明确做什么，做成什么样

| 章节 | 内容概览 |
|------|---------|
| 1. 项目概述 | 项目背景、目标 |
| 2. 用户角色 | 普通员工、负责人 |
| 3. 功能需求 | 三条核心链路详细说明 + 验收标准 |
| 4. 页面设计 | 页面清单、聊天页布局、工单页布局 |
| 5. 非功能性需求 | 响应时间、并发等指标 |
| 6. 验收标准总览 | 四个阶段的验收清单 |
| 7. 项目范围边界 | 包含/不包含的功能（防过度开发） |

### 2. 架构文档 (`02-架构文档.md`)
