## 各文档详情

### 1. 需求文档 (`01-需求文档.md`)

**核心用途**: 明确做什么，做成什么样

| 章节 | 内容概览 |
|------|---------|
| 1. 项目概述 | 项目背景、目标 |
| 2. 用户角色 | 普通员工、负责人 |
| 3. 功能需求 | 三条核心链路详细说明 + 验收标准 |
| 4. 页面设计 | 页面清单、聊天页布局、工单页布局 |
| 5. 非功能性需求 | 响应时间、并发等指标 |
| 6. 验收标准总览 | 四个阶段的验收清单 |
| 7. 项目范围边界 | 包含/不包含的功能（防过度开发） |
