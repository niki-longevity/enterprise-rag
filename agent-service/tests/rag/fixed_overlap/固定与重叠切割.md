# 办公行为规范

## 1. 总则

### 1.1 目的
为维护公司良好的办公秩序，营造和谐、高效的工作环境，规范员工日常办公行为，提升企业整体形象，特制定本规范。本规范适用于公司全体员工，包括正式员工、试用期员工、实习生及外包人员。所有进入公司办公区域的人员均应严格遵守本规范的各项规定。

### 1.2 适用范围
本规范适用于公司所有办公区域，包括但不限于：总部办公楼、各分公司办公室、外派机构办公场所、公司指定的临时办公地点等。员工在外出办公、参加商务活动时，也应参照本规范的相关要求，维护公司形象。

### 1.3 责任部门
人力资源部是本规范的归口管理部门，负责规范的制定、修订、解释和监督执行。各部门负责人负责本部门员工对本规范的宣导和落实，发现违规行为应及时纠正。行政部负责办公环境的日常维护和相关设施的保障。

## 2. 考勤管理

### 2.1 工作时间
- **标准工作时间**：周一至周五，9:00-18:00，每天工作8小时
- **午休时间**：12:00-13:00，共计1小时，员工可在此期间用餐、休息
- **弹性工作制**：公司实行弹性工作制，核心工作时间10:00-16:00必须在岗，其余时间可根据个人工作安排适当调整，但需确保每日工作满8小时
- **早到/晚走**：鼓励员工根据工作需要灵活安排时间，早到或晚走需确保不影响其他同事的正常休息
- **周末加班**：原则上不鼓励周末加班，确因工作需要加班的，需提前申请并经部门主管批准，可安排调休或按国家规定支付加班费

### 2.2 打卡管理
- **打卡方式**：使用公司统一的考勤系统进行打卡，支持指纹、人脸识别或手机APP打卡
- **打卡次数**：每日需打卡两次，上班打卡（9:00前）和下班打卡（18:00后）
- **忘记打卡**：如因特殊原因忘记打卡，需在3个工作日内提交《未打卡说明》，经部门主管签字确认后报备人力资源部
- **代打卡**：严禁代打卡行为，一经发现，代打卡者和委托代打卡者双方均按严重违纪处理，给予记过处分并扣除当月全勤奖
- **外出打卡**：因公外出不能按时打卡的，需提前在系统中提交外出申请，并经部门主管批准

### 2.3 迟到早退
- **迟到/早退30分钟以内**：给予口头警告，当月累计不超过2次
- **迟到/早退30分钟以上**：按事假处理，扣除相应时长的工资
- **迟到/早退2小时以上**：按旷工半天处理
- **月度累计3次以上**：扣除全勤奖，并取消当年评优资格
- **年度累计10次以上**：视为严重违反公司规章制度，公司有权解除劳动合同

### 2.4 旷工处理
- **旷工半天**：扣除1.5倍日薪
- **旷工1天**：扣除3倍日薪
- **连续旷工3天或年度累计旷工5天**：视为严重违纪，公司有权解除劳动合同，无需支付经济补偿金
- **旷工认定**：未履行请假手续或请假未获批准而擅自不到岗的，视为旷工

## 3. 办公环境

### 3.1 工位管理
- **保持工位整洁**：每日下班前整理桌面，文件资料归类存放，保持工位干净有序
- **禁止在办公区域进食异味食物**：办公区域禁止食用韭菜、大蒜、榴莲等异味食物，可在茶水间或指定用餐区域用餐
- **个人物品收纳**：个人物品应收纳在指定区域，不得占用公共通道或他人工位
- **绿植管理**：工位可摆放小型绿植，但需定期养护，避免干枯或滋生病虫害
- **电器使用**：下班后关闭电脑、显示器、台灯等电器设备电源，节约能源
- **贵重物品**：个人贵重物品请妥善保管，如有遗失公司不承担责任

### 3.2 宠物政策
- **禁止携带宠物进入办公区域**：包括但不限于办公室、会议室、走廊、电梯等公共区域

## 4. 会议纪律

### 4.1 会议准时
- **提前5分钟进入会议室**：参会人员应提前5分钟进入会议室，做好会议准备
- **准时开始**：会议主持人应准时开始会议，不等待迟到人员
- **迟到处理**：迟到5分钟以内的，会后向主持人说明原因；迟到10分钟以上的，需向会议组织部门提交书面说明
- **会议请假**：确实无法参加会议的，需提前向会议主持人请假，并安排合适人员代为参加

### 4.2 会议礼仪
- **手机调至静音**：会议期间手机应调至静音或震动模式，禁止在会议室内接听电话
- **紧急情况处理**：如有紧急电话需接听，应轻声离开会议室，到外面接听
- **发言秩序**：发言应先举手示意，经主持人同意后再发言，不随意打断他人发言
- **发言时长**：发言应简明扼要，控制好时间，不长篇大论
- **会议记录**：会议组织部门应安排专人做好会议记录，会后24小时内将会议纪要发送给所有参会人员

### 4.3 会场秩序
- **非紧急情况不得中途离席**：会议期间应保持专注，不擅自离席，确需离席的应向主持人说明
- **禁止私下交谈**：会议期间禁止私下交谈，有问题可在提问环节提出
- **会场设施爱护**：爱护会议室的桌椅、投影仪、白板等设施，使用完毕及时归位
- **会后清理**：会议结束后，参会人员应整理好自己的物品，带走垃圾，保持会场整洁
- **会议室预约**：使用会议室需提前在系统中预约，未经预约不得占用

## 5. 着装规范

### 5.1 日常着装
- **工作日着装**：周一至周四应穿着商务休闲装，男士可穿衬衫、Polo衫、休闲西裤，女士可穿衬衫、套裙、连衣裙等
- **周五着装**：周五可穿着便装，但应保持整洁得体，不得穿着过于随意
- **着装禁忌**：禁止穿着背心、短裤、拖鞋、超短裙、透视装等不适宜办公的服装
- **特殊岗位**：前台、客户服务等对外窗口岗位应穿着正装

### 5.2 正式场合着装
- **重要会议**：参加重要会议、会见重要客户时，应穿着正装
- **男士正装**：西装、衬衫、领带、皮鞋
- **女士正装**：西装套裙或套装、衬衫、皮鞋
- **着装整洁**：正装应保持干净、平整、无褶皱
- **配饰简洁**：配饰应简洁大方，不佩戴过于夸张的饰品

## 6. 沟通规范

### 6.1 内部沟通
- **工作沟通渠道**：工作沟通应优先使用公司指定的即时通讯工具、邮件或项目管理系统
- **响应时效**：收到工作消息后，应在2小时内予以回复，紧急事项应立即回复
- **邮件规范**：邮件应主题明确、内容简洁、条理清晰，重要邮件应抄送给相关人员
- **称呼礼貌**：沟通时应使用礼貌用语，尊重同事，不使用侮辱性、攻击性语言
- **尊重他人**：尊重他人的工作习惯和个人隐私，不随意翻看他人文件、电脑

### 6.2 对外沟通
- **客户沟通**：与客户沟通应耐心、细致、专业，维护公司良好形象
- **商务礼仪**：会见客户时应遵守商务礼仪，提前到达，主动握手，交换名片
- **保密原则**：对外沟通时应遵守公司保密制度，不泄露公司商业秘密
- **用语规范**：使用规范、专业的用语，不随意承诺超出权限的事项
- **及时汇报**：对外沟通中遇到重要情况应及时向主管汇报

## 7. 附则

### 7.1 规范执行
- 本规范自发布之日起正式执行，原有相关规定与本规范不一致的，以本规范为准
- 人力资源部有权根据实际情况对本规范进行修订，修订后的规范将另行通知
- 各部门应组织员工认真学习本规范，确保全员知晓、自觉遵守

### 7.2 违规处理
- 轻微违规：首次违反本规范的，给予口头警告，责令立即改正
- 一般违规：再次违反或违反情节较重的，给予通报批评，扣除当月绩效奖金10%
- 严重违规：多次违反或违反情节严重的，给予记过处分，扣除当月绩效奖金30%，取消当年评优资格
- 特别严重违规：违反情节特别严重，给公司造成重大损失或恶劣影响的，公司有权解除劳动合同，并追究相关法律责任

### 7.3 意见反馈
- 员工对本规范有任何意见或建议，可通过以下渠道反馈：人力资源部邮箱、意见箱、员工代表大会
- 人力资源部将认真对待每一条意见和建议，对合理的意见和建议将积极采纳
- 公司每年将对本规范的执行情况进行评估，根据评估结果和员工反馈进行修订完善

# 设备管理办法

## 1. 总则

### 1.1 目的
为规范公司办公设备的管理，提高设备使用效率，延长设备使用寿命，确保设备安全运行，保障公司各项工作的顺利开展，特制定本办法。本办法适用于公司所有办公设备、IT设备、会议设备及相关耗材的采购、领用、使用、维护、报废等全生命周期管理。

### 1.2 适用范围
本办法适用于公司全体员工及各部门。本办法所指的设备包括但不限于：
- **IT设备**：台式电脑、笔记本电脑、平板电脑、服务器、网络设备等
- **办公设备**：打印机、复印机、扫描仪、传真机、电话机等
- **会议设备**：投影仪、音响系统、视频会议系统、电子白板等
- **移动设备**：手机、移动硬盘、U盘等
- **其他设备**：根据工作需要配置的其他专业设备

- **IT部门**：负责IT设备、网络设备的采购、配置、维护、维修、报废等管理工作

- **申请流程**：

2. 部门主管审核，确认申请的合理性和必要性
3. IT部门或行政部复核，根据设备库存情况和配置标准进行审核
4. 分管领导审批（超过标准配置的需总经理审批）
5. 审批通过后，IT部门或行政部安排设备采购或调配
6. 员工到IT部门或行政部办理领用手续，签字确认

- **申请时间**：
- 新员工入职设备申请：应在入职前3个工作日提交申请
- 日常设备申请：应提前5个工作日提交申请
- 紧急设备申请：可向IT部门或行政部申请加急处理，但需说明紧急原因

- **注意事项**：
- 申请设备应优先考虑调配现有库存设备，确需新购的再进行采购
- 设备型号应优先选择公司协议供应商的产品，确保质量和售后服务
- 设备申请批准后，应在10个工作日内办理领用手续，逾期未领用的视为自动放弃

### 2.3 更换流程
- **更换条件**：
- 设备使用满3年，性能无法满足工作需要的
- 设备出现故障，维修费用超过设备残值50%的
- 设备损坏无法修复的
- 因岗位调整需要更换设备的
- 公司统一更新设备的

- **更换流程**：
1. 员工填写《设备更换申请表》，说明更换原因，附相关证明材料
2. 部门主管审核，确认更换的必要性
3. IT部门或行政部进行技术鉴定，评估设备状况
4. 分管领导审批
5. 审批通过后，员工交回旧设备，领取新设备
6. IT部门或行政部办理设备变更登记，更新设备台账

- **更换周期**：
- 一般设备更换：7个工作日内完成
- 特殊设备更换：15个工作日内完成
- 公司统一更换：按公司通知的时间安排执行

- **旧设备处理**：
- 交回的旧设备由IT部门或行政部统一处理
- 仍有使用价值的设备，将进行清洁、检测、维修后入库备用
- 无法继续使用的设备，按报废流程处理
- 设备中的数据应在交回前由员工自行备份，IT部门不负责数据恢复

## 3. 设备使用

### 3.1 日常维护
- **定期备份重要数据**：
- 员工应定期备份重要工作数据，建议每周至少备份一次
- 重要数据应备份到公司指定的文件服务器或云存储
- 禁止将公司重要数据存储到个人设备或第三方云盘
- IT部门会定期检查数据备份情况，提醒员工及时备份

- **禁止安装非授权软件**：
- 禁止在公司设备上安装未经授权的软件
- 确因工作需要安装软件的，应向IT部门申请，由IT部门统一安装
- 禁止安装盗版软件、破解软件、游戏软件等与工作无关的软件
- 禁止使用公司设备进行挖矿、刷单、炒币等活动

- **定期进行系统更新**：
- 应及时安装系统安全更新和补丁，防范安全漏洞
- IT部门会定期推送系统更新，员工应及时安装
- 更新前应备份重要数据，避免数据丢失
- 如更新后出现问题，应及时联系IT部门处理

- **保持设备清洁**：

# 休假管理规定

## 1. 总则

### 1.1 目的
为保障员工的休息权利，规范公司休假管理，维护公司正常的工作秩序，根据《中华人民共和国劳动法》、《中华人民共和国劳动合同法》及国家和地方相关法律法规，结合公司实际情况，特制定本规定。本规定适用于公司全体员工的各类休假管理。

### 1.2 适用范围
本规定适用于公司全体正式员工、试用期员工。实习生、劳务派遣人员的休假参照本规定执行，具体按实习协议或劳务派遣协议约定执行。公司的休假类型包括：带薪年假、病假、事假、婚假、产假、陪产假、丧假、工伤假等国家规定的各类假期。

### 1.3 管理职责
- **人力资源部**：负责休假制度的制定、修订、解释和监督执行，负责各类休假的备案和统计
- **各部门负责人**：负责本部门员工休假申请的审核，合理安排工作，确保业务正常开展
- **员工本人**：负责按规定申请休假，提前做好工作交接，休假期间保持通讯畅通
- **行政部**：负责员工考勤记录的管理，配合人力资源部做好休假统计工作

## 6. 考核等级

### 6.1 等级划分
- **A（优秀）**：绩效表现突出，远超预期目标
- 标准：目标完成率≥120%，工作质量高，创新性强
- 比例：不超过员工总数的15%
- **B（良好）**：绩效表现良好，达到或超过预期目标
- 标准：目标完成率100%-119%，工作质量良好
- 比例：不超过员工总数的30%
- **C（合格）**：绩效表现合格，基本达到预期目标
- 标准：目标完成率80%-99%，工作质量合格
- 比例：员工总数的40%-50%
- **D（待改进）**：绩效表现未达到预期目标
- 标准：目标完成率60%-79%，工作质量有待提高
- 比例：不超过员工总数的10%
- **E（不合格）**：绩效表现差，远未达到预期目标
- 标准：目标完成率<60%，工作质量差
- 比例：不超过员工总数的5%

### 6.2 等级应用
- **A级**：年度绩效奖金上浮50%，优先晋升、培训机会
- **B级**：年度绩效奖金上浮20%，适当晋升、培训机会
- **C级**：年度绩效奖金标准发放，常规培训机会
- **D级**：年度绩效奖金下浮30%，制定改进计划，安排培训
- **E级**：年度绩效奖金取消，制定改进计划，观察期或调整岗位

### 6.3 等级分布
- 各部门应合理分布考核等级
- 避免平均主义或过度集中
- 不同等级应有合理的比例
- 特殊情况需说明理由

### 1.1 目的
为规范公司招聘与入职管理，确保招聘过程公正、高效，吸引和选拔优秀人才，保障新员工顺利入职，根据公司人力资源管理需要，特制定本流程。本流程适用于公司所有岗位的招聘和入职管理。
