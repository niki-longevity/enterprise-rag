# [41] 我出差回来后发现借用的投影仪遥控器丢了，设备外壳也有划痕，请问这种情况需要赔偿吗？赔偿金额怎么计算？另外出差期间我因为紧急情况乘坐了出租车，票据丢失了还能报销吗？这种违规情况会被记录到我的个人档案吗？

## Agent 回答

根据公司相关政策，我来为您详细解答：

## 一、关于投影仪遥控器丢失和设备划痕的赔偿问题

**需要赔偿的情况：**
根据《设备管理办法》第4.3条规定：
- **配件丢失**：遥控器属于设备配件，按配件市场价格赔偿
- **设备损坏**：设备外壳划痕属于人为损坏，需要赔偿

**赔偿金额计算方式：**
1. **遥控器丢失**：按配件市场价格赔偿（需向行政部或IT部门查询具体市场价格）
2. **设备外壳划痕**：
   - 如果是**部分损坏**：按维修费用赔偿
   - 如果是**人为损坏**：按设备折旧价格赔偿
   - **折旧计算**：使用第1年折旧20%，第2年折旧15%，第3年及以后每年折旧10%

**赔偿流程：**
1. 管理员评估损坏情况，确定赔偿金额
2. 向您发出《赔偿通知书》
3. 您在7个工作日内缴纳赔偿金
4. 对赔偿金额有异议的，可向行政部或IT部门申请复核

**特别提醒：**
- 如果设备价值超过5000元且借用时缴纳了押金，将从押金中扣除维修或赔偿费用
- 故意损坏公司设备的，除赔偿外，还将给予纪律处分

## 二、关于出租车票据丢失的报销问题

**可以报销，但需要提供替代凭证：**
根据《差旅报销制度》第3.3条规定：
- **市内交通报销**：地铁、公交等没有票据的，可凭**支付记录报销**
- **出租车报销**：应保存好发票，但如票据丢失，可参考公共交通的处理方式

**具体操作建议：**
1. **提供支付记录**：如果使用电子支付（微信、支付宝、银行卡等），可提供支付凭证截图
2. **书面说明**：详细说明乘坐出租车的时间、起止地点、事由（紧急情况）
3. **部门主管确认**：请部门主管在报销单上签字确认情况的真实性
4. **报销时限**：出差结束后7个工作日内提交报销申请

**注意事项：**
- 出租车乘坐应符合规定情形（如时间紧急、公共交通不便等）
- 单次出租车费超过100元的，应提前向部门主管说明
- 报销时应注明起止地点和事由

## 三、关于违规记录和个人档案问题

**通常不会记录到个人档案：**
根据搜索到的政策文档：
1. **违规处理分级**：
   - 轻微违规：口头警告，责令立即改正
   - 一般违规：通报批评，扣除当月绩效奖金10%
   - 严重违规：记过处分，扣除当月绩效奖金30%，取消当年评优资格
   - 特别严重违规：解除劳动合同

2. **您的情况分析**：
   - **设备赔偿**：属于正常的工作设备损坏/丢失赔偿，不属于违规处罚
   - **票据丢失报销**：只要如实说明情况并提供替代凭证，属于正常报销流程，不构成违规

3. **会被记录的情况**：
   - 只有达到"一般违规"及以上级别的违纪行为才会被记录
   - 故意损坏设备、虚假报销等行为才会被纪律处分
   - 正常的设备赔偿和票据补报不属于违纪行为

**建议：**
- 主动联系行政部/IT部门报告设备情况，配合完成赔偿流程
- 向财务部说明出租车票据丢失情况，提供支付记录和书面说明
- 保持沟通透明，这类工作失误通常不会影响个人档案

如有其他疑问，建议直接咨询行政部（设备赔偿）和财务部（报销事宜）。

## 4. 资源借用

### 4.1 可借用设备
- **会议设备**：
- 投影仪、投影幕布
- 摄像机、照相机、录音笔
- 便携音响、麦克风
- 视频会议设备

- **IT设备**：
- 备用笔记本电脑
- 备用显示器
- 移动硬盘、U盘等存储设备
- 网络测试设备

- **办公设备**：
- 便携打印机、扫描仪
- 计算器、订书机等小型办公设备
- 其他可移动的办公设备

### 4.2 借用规则
- **借用申请**：
- 提前1天在设备管理系统中提交借用申请
- 注明借用设备名称、借用时间、归还时间、借用用途
- 部门主管审批（借用时间超过3天的需分管领导审批）
- 审批通过后，到行政部或IT部门办理借用手续，签字确认

- **借用期限**：
- 一般设备借用期限不超过3个工作日
- 特殊情况需要延长借用期限的，应提前申请续借
- 续借期限不超过原借用期限
- 累计借用期限不超过10个工作日

- **借用押金**：
- 价值超过5000元的设备借用时需缴纳押金
- 押金金额为设备价值的20%-50%
- 归还设备时如无损坏，押金全额退还
- 设备损坏的，从押金中扣除维修费用或赔偿费用

- **逾期处理**：
- 逾期未还设备，影响信用评分
- 逾期超过3天的，暂停设备借用资格3个月
- 逾期超过7天的，按设备价值的0.5%/天收取占用费
- 逾期超过15天的，视为丢失，按设备全价赔偿

### 4.3 归还与损坏赔偿
- **设备归还**：
- 按时归还设备，归还时应保持设备完好、清洁
- 管理员检查设备状况，确认无误后办理归还手续
- 如有损坏或配件丢失，应在《设备归还单》上注明
- 归还后管理员更新设备状态，办理押金退还手续

- **损坏赔偿**：
- 人为损坏设备：按设备折旧价格赔偿
- 设备折旧计算：使用第1年折旧20%，第2年折旧15%，第3年及以后每年折旧10%
- 部分损坏：按维修费用赔偿
- 配件丢失：按配件市场价格赔偿

- **赔偿流程**：
- 管理员评估损坏情况，确定赔偿金额
- 向责任人发出《赔偿通知书》
- 责任人在7个工作日内缴纳赔偿金
- 对赔偿金额有异议的，可向行政部或IT部门申请复核
