# 设备管理办法

## 1. 总则

### 1.1 目的
为规范公司办公设备的管理，提高设备使用效率，延长设备使用寿命，确保设备安全运行，保障公司各项工作的顺利开展，特制定本办法。本办法适用于公司所有办公设备、IT设备、会议设备及相关耗材的采购、领用、使用、维护、报废等全生命周期管理。

### 1.2 适用范围
本办法适用于公司全体员工及各部门。本办法所指的设备包括但不限于：
- **IT设备**：台式电脑、笔记本电脑、平板电脑、服务器、网络设备等
- **办公设备**：打印机、复印机、扫描仪、传真机、电话机等
- **会议设备**：投影仪、音响系统、视频会议系统、电子白板等
- **移动设备**：手机、移动硬盘、U盘等
- **其他设备**：根据工作需要配置的其他专业设备

### 1.3 管理职责
- **IT部门**：负责IT设备、网络设备的采购、配置、维护、维修、报废等管理工作
- **行政部**：负责办公设备、会议设备的采购、登记、调拨、报废等管理工作
- **各部门**：负责本部门设备的日常使用、保管和维护，配合IT部门和行政部做好设备管理工作
- **使用人**：负责所领用设备的日常保管、合理使用和维护，确保设备安全和完好

## 2. 设备领用

### 2.1 标准配置
- **新员工入职标配**：
  - 笔记本电脑一台（根据岗位配置不同型号）
  - 显示器一台（23寸或以上）
  - 键盘、鼠标一套
  - 办公用品一套（按需发放）

- **岗位特殊需求**：
  - 设计岗位：可申请高配电脑、绘图板、专业显示器等
  - 开发岗位：可申请高配电脑、多显示器、测试设备等
  - 视频岗位：可申请摄像机、麦克风、补光灯等专业设备
  - 管理岗位：可申请手机、平板电脑等移动办公设备

- **配置标准**：人力资源部会同IT部门、行政部根据不同岗位的工作需要，制定《设备配置标准表》，并定期更新

### 2.2 设备申请
- **申请流程**：
  1. 员工填写《设备领用申请表》，注明申请设备的名称、型号、用途等
  2. 部门主管审核，确认申请的合理性和必要性
  3. IT部门或行政部复核，根据设备库存情况和配置标准进行审核
  4. 分管领导审批（超过标准配置的需总经理审批）
  5. 审批通过后，IT部门或行政部安排设备采购或调配
  6. 员工到IT部门或行政部办理领用手续，签字确认

- **申请时间**：
  - 新员工入职设备申请：应在入职前3个工作日提交申请
  - 日常设备申请：应提前5个工作日提交申请
  - 紧急设备申请：可向IT部门或行政部申请加急处理，但需说明紧急原因

- **注意事项**：
  - 申请设备应优先考虑调配现有库存设备，确需新购的再进行采购
  - 设备型号应优先选择公司协议供应商的产品，确保质量和售后服务
  - 设备申请批准后，应在10个工作日内办理领用手续，逾期未领用的视为自动放弃

### 2.3 更换流程
- **更换条件**：
  - 设备使用满3年，性能无法满足工作需要的
  - 设备出现故障，维修费用超过设备残值50%的
  - 设备损坏无法修复的
  - 因岗位调整需要更换设备的
  - 公司统一更新设备的

- **更换流程**：
  1. 员工填写《设备更换申请表》，说明更换原因，附相关证明材料
  2. 部门主管审核，确认更换的必要性
  3. IT部门或行政部进行技术鉴定，评估设备状况
  4. 分管领导审批
  5. 审批通过后，员工交回旧设备，领取新设备
  6. IT部门或行政部办理设备变更登记，更新设备台账

- **更换周期**：
  - 一般设备更换：7个工作日内完成
  - 特殊设备更换：15个工作日内完成
  - 公司统一更换：按公司通知的时间安排执行

- **旧设备处理**：
  - 交回的旧设备由IT部门或行政部统一处理
  - 仍有使用价值的设备，将进行清洁、检测、维修后入库备用
  - 无法继续使用的设备，按报废流程处理
  - 设备中的数据应在交回前由员工自行备份，IT部门不负责数据恢复

## 3. 设备使用

### 3.1 日常维护
- **定期备份重要数据**：
  - 员工应定期备份重要工作数据，建议每周至少备份一次
  - 重要数据应备份到公司指定的文件服务器或云存储
  - 禁止将公司重要数据存储到个人设备或第三方云盘
  - IT部门会定期检查数据备份情况，提醒员工及时备份

- **禁止安装非授权软件**：
  - 禁止在公司设备上安装未经授权的软件
  - 确因工作需要安装软件的，应向IT部门申请，由IT部门统一安装
  - 禁止安装盗版软件、破解软件、游戏软件等与工作无关的软件
  - 禁止使用公司设备进行挖矿、刷单、炒币等活动

- **定期进行系统更新**：
  - 应及时安装系统安全更新和补丁，防范安全漏洞
  - IT部门会定期推送系统更新，员工应及时安装
  - 更新前应备份重要数据，避免数据丢失
  - 如更新后出现问题，应及时联系IT部门处理

- **保持设备清洁**：
  - 定期清洁设备表面，保持设备整洁
  - 禁止在设备附近喝水、吃东西，防止液体溅入设备
  - 显示器、键盘、鼠标等应定期清洁，防止灰尘积累
  - 设备发生液体泼溅等意外情况时，应立即关机并联系IT部门处理

### 3.2 安全管理
- **设置开机密码**：
  - 所有办公设备必须设置开机密码
  - 密码长度至少8位，包含大小写字母、数字和特殊字符
  - 密码应每90天更换一次，不得重复使用最近5次的密码
  - 密码应妥善保管，不得告诉他人，不得写在纸上贴在设备上

- **锁屏设置**：
  - 离开工位时应锁定电脑屏幕，防止他人操作
  - 电脑应设置自动锁屏，闲置超过10分钟自动锁定
  - 下班时应关闭电脑及相关设备电源
  - 笔记本电脑应妥善保管，不得随意放置在办公桌上

- **网络安全**：
  - 禁止连接未授权的WiFi网络，只能使用公司提供的网络
  - 禁止使用公司设备访问非法网站、色情网站、赌博网站等
  - 禁止下载来源不明的文件，防止病毒感染
  - 发现设备异常（如运行缓慢、弹出异常窗口等）应及时联系IT部门

- **数据保密**：
  - 禁止将公司内部数据、商业秘密等上传到互联网
  - 禁止使用私人邮箱、微信、QQ等传输公司敏感文件
  - 敏感文件应使用公司指定的加密方式传输
  - 离职时应删除公司设备上的个人数据，交接所有工作文件

### 3.3 故障处理
- **故障报修**：
  - 设备出现故障时，应及时通过IT服务台或报修系统提交报修申请
  - 报修时应详细描述故障现象、故障发生时间、操作步骤等信息
  - 紧急故障可直接联系IT部门值班人员
  - 报修后应保持通讯畅通，配合IT部门进行故障排查

- **响应时间**：
  - 一般故障：IT部门应在4小时内响应，24小时内修复
  - 紧急故障：IT部门应在30分钟内响应，4小时内修复
  - 复杂故障：IT部门应及时告知预计修复时间，必要时提供备用设备
  - 无法现场修复的设备，IT部门应安排送修或更换

- **维修注意事项**：
  - 设备维修期间，IT部门会尽力保护数据安全，但不承诺数据不丢失
  - 送修前员工应备份重要数据，IT部门不负责数据备份
  - 维修费用在保修期内的由供应商承担，超过保修期的由公司承担
  - 人为损坏的维修费用由责任人承担

## 4. 资源借用

### 4.1 可借用设备
- **会议设备**：
  - 投影仪、投影幕布
  - 摄像机、照相机、录音笔
  - 便携音响、麦克风
  - 视频会议设备

- **IT设备**：
  - 备用笔记本电脑
  - 备用显示器
  - 移动硬盘、U盘等存储设备
  - 网络测试设备

- **办公设备**：
  - 便携打印机、扫描仪
  - 计算器、订书机等小型办公设备
  - 其他可移动的办公设备

### 4.2 借用规则
- **借用申请**：
  - 提前1天在设备管理系统中提交借用申请
  - 注明借用设备名称、借用时间、归还时间、借用用途
  - 部门主管审批（借用时间超过3天的需分管领导审批）
  - 审批通过后，到行政部或IT部门办理借用手续，签字确认

- **借用期限**：
  - 一般设备借用期限不超过3个工作日
  - 特殊情况需要延长借用期限的，应提前申请续借
  - 续借期限不超过原借用期限
  - 累计借用期限不超过10个工作日

- **借用押金**：
  - 价值超过5000元的设备借用时需缴纳押金
  - 押金金额为设备价值的20%-50%
  - 归还设备时如无损坏，押金全额退还
  - 设备损坏的，从押金中扣除维修费用或赔偿费用

- **逾期处理**：
  - 逾期未还设备，影响信用评分
  - 逾期超过3天的，暂停设备借用资格3个月
  - 逾期超过7天的，按设备价值的0.5%/天收取占用费
  - 逾期超过15天的，视为丢失，按设备全价赔偿

### 4.3 归还与损坏赔偿
- **设备归还**：
  - 按时归还设备，归还时应保持设备完好、清洁
  - 管理员检查设备状况，确认无误后办理归还手续
  - 如有损坏或配件丢失，应在《设备归还单》上注明
  - 归还后管理员更新设备状态，办理押金退还手续

- **损坏赔偿**：
  - 人为损坏设备：按设备折旧价格赔偿
  - 设备折旧计算：使用第1年折旧20%，第2年折旧15%，第3年及以后每年折旧10%
  - 部分损坏：按维修费用赔偿
  - 配件丢失：按配件市场价格赔偿

- **赔偿流程**：
  - 管理员评估损坏情况，确定赔偿金额
  - 向责任人发出《赔偿通知书》
  - 责任人在7个工作日内缴纳赔偿金
  - 对赔偿金额有异议的，可向行政部或IT部门申请复核

## 5. 设备报废

### 5.1 报废条件
- 达到使用年限，且性能无法满足工作需要的
- 设备损坏，无法修复或维修费用超过设备残值的
- 技术落后，不符合工作要求且无升级价值的
- 国家规定必须淘汰的高耗能、高污染设备
- 因公司业务调整不再需要的设备

### 5.2 报废流程
- 设备使用部门提出报废申请，填写《设备报废申请表》
- IT部门或行政部进行技术鉴定，出具鉴定意见
- 财务部进行资产核算，确定设备残值
- 分管领导审核，总经理审批
- 审批通过后，IT部门或行政部办理设备注销手续
- 财务部进行账务处理，核销固定资产
- 行政部负责报废设备的处置

### 5.3 报废处置
- 仍有使用价值的设备，可捐赠给公益机构或低价转让给员工
- 无使用价值但有回收价值的设备，出售给有资质的回收公司
- 无回收价值的设备，按环保要求进行拆解处理
- 存储设备报废前应进行数据销毁，防止数据泄露
- 报废处置收入上缴公司财务部

## 6. 附则

### 6.1 办法执行
- 本办法自发布之日起施行，原《设备管理办法》同时废止
- IT部门和行政部负责本办法的解释和修订
- 每年对本办法的执行情况进行评估，根据需要进行修订

### 6.2 责任追究
- 违反本办法规定，造成设备损坏或丢失的，按规定赔偿
- 故意损坏公司设备的，除赔偿外，还将给予纪律处分
- 侵占公司设备的，视为严重违纪，公司有权解除劳动合同，并追究法律责任

### 6.3 其他
- 各部门应指定专人负责本部门设备的日常管理
- IT部门和行政部应建立设备台账，定期盘点，做到账实相符
- 公司每年进行一次设备全面盘点，掌握设备使用状况
- 员工应爱护公司设备，节约使用，提高设备使用效率
